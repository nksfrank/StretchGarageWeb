Första uppstart

När appen startas upp första gången presenteras appens profilvy. För att komma igång och använda tjänsten fyller du i fälten Ditt Namn och Ditt Telefonnummer (telefonnumret används endast som ett unikt ID för att hålla koll på din profil). Där efter klickar du på SKICKA. Din profil registreras i tjänsten och du tas vidare till appens startsida.

Navigera i appen

Appens startsida

På appens startsida listas samtliga parkeringsplatser. Denna vyn överblickar parkeringsplatsernas namn och totala antalet platser.

Detaljerad parkeringsvy

Tryck eller klicka på ett garagenamn i appens startsida för att se en detaljerad vy över en parkeringsplats. Den detaljerade vyn presenterar lediga (gröna) och upptagna (röda) platser.

Backa

Tappa eller klicka på Backa för att när som helst stega tillbaka i applikationens vyer.

Funktioner

Din Profil

För att komma åt din profil och ändra inställningar i appen

  1. Tappa eller klicka på meny-knappen () för att öppna menyn.
  2. Tappa eller klicka på Profil i menylistan.

Stoppa GPS

GPS funktionen är igång när applikationen installeras. Ifall du känner att denna funktion inte fungerar som den ska kan du stänga av denna funktionen i inställningarna under din profil.

  1. Tappa eller klicka på meny-knappen () för att öppna menyn
  2. Tappa eller klicka på Profil i menylistan
  3. Tappa eller klicka på knappen Stoppa GPS

Starta GPS

Ifall GPS funktionen har stoppats kan du starta denna funktionen i inställningarna under din profil.

  1. Tappa eller klicka på meny-knappen () för att öppna menyn
  2. Tappa eller klicka på Profil i menylistan
  3. Tappa eller klicka på knappen Starta GPS

Manuell Parkering

Ifall något skulle gå fel med den automatiserade parkeringsprocessen kan du parkera manuellt.

  1. Navigera in på den detaljerade parkeringsvyn i fråga
  2. Tappa eller klicka på meny-knappen () för att öppna menyn
  3. Tappa eller klicka på Manuell parkering i menylistan
Du har parkerats
Ett meddelande presenteras överst på skärmen när parkeringen lyckas (ett rött felmeddelande kan komma att presenteras ifall ett fel har uppståt i tjänsten).

Manuell Avparkering

Ifall något skulle gå fel med den automatiserade avparkeringsprocessen kan du avparkera manuellt.

  1. Tappa eller klicka på meny-knappen () för att öppna menyn
  2. Tappa eller klicka på Manuell avparkering i menylistan
Du har avparkerats
Ett meddelande presenteras överst på skärmen när parkeringen lyckas (ett rött felmeddelande kan komma att presenteras ifall ett fel har uppståt i tjänsten).
